🚀 MERN Stack & React Practice
A curated collection of projects and exercises focused on mastering full-stack development, advanced React patterns, and modern styling. This repository tracks my progress through the MERN ecosystem.

📁 Repository Structure
The repository is organized by technology and complexity:

1-Foundations/ – Core JavaScript and HTML/CSS concepts.

2-Node-Express/ – Backend logic and API development.

3-React/ * 19-todo-app use-reducer/ – A robust Todo application utilizing the useReducer hook for complex state management.

4-MongoDB/ – Database schemas and integration.

🛠️ Tech Stack
Frontend: React.js, Tailwind CSS

State Management: Hooks (useState, useReducer, useContext)

Backend: Node.js, Express.js

Database: MongoDB

Tools: Git, Vite, Postman

🔥 Featured Project: Todo App (useReducer)
Located in 3 React/19-todo-app use-reducer/, this project demonstrates a transition from simple useState to a more scalable useReducer pattern.

Key Features
State Logic: Centralized action handling (Add, Delete, Toggle, Clear).

Responsive UI: Styled with Tailwind CSS for a clean, modern look.

Persistence: (Optional: Mention if you used LocalStorage).
